Installation & Setup

Guide for setting up the 3dverse development environment locally.

Prerequisites

Before installing, ensure your system meets the following requirements:

  • Node.js (v18 or higher)
    Required to run the build tools and local development server.
  • Gemini API Key
    Required for the AI Assistant features. Get one at Google AI Studio.
  • Git
    To clone the repository and manage updates.

Developer Install (Unpacked)

Follow these steps to run the application source code on your local machine.

Terminal (Universal)

1. Clone the Repository

Download the source code to your local machine.

git clone https://github.com/your-org/3dverse-collaborate.git cd 3dverse-collaborate

2. Install Dependencies

Install the required packages listed in package.json.

npm install

3. Configure Environment

Create a .env file in the root directory to store your secrets. Do not commit this file.

# Create .env file touch .env # Add your API Key echo "API_KEY=your_gemini_api_key_here" >> .env
Security Note: In this local demo, the API key is accessed via process.env. In a production build, ensure keys are handled via a backend proxy to prevent exposure.

4. Run Application

Start the local development server.

npm start

The application should automatically open at http://localhost:3000.

[Screenshot: Dashboard View loaded in localhost]

Security & Permissions

Upon launching the application for the first time, you will be prompted to grant the following permissions:

Microphone Access

Required for:

  • Real-time voice chat with other collaborators.
  • Analyzing audio levels to show "who is speaking" indicators.
You can revoke this permission at any time via your browser's site settings (Lock icon in URL bar).

Troubleshooting

Common issues encountered during installation:

Port 3000 is busy

If the default port is occupied, the start script usually prompts to use another port (e.g., 3001). You can also force a port:

PORT=3001 npm start

"Missing API Key" Error

If the AI features (Sparkles icon) return an error or loader spins indefinitely:

  1. Verify .env file exists in the root.
  2. Ensure the variable name is exactly API_KEY.
  3. Restart the development server (`Ctrl+C`, then `npm start`) to load new env vars.

Microphone not working

Check if your browser has blocked access globally. Go to chrome://settings/content/microphone to verify.

Updating & Uninstalling

Updating

To pull the latest changes from the repository:

git pull origin main npm install

Uninstalling

To remove the application, simply delete the project folder:

rm -rf 3dverse-collaborate
