Introduction

3dverse Collaborate is a browser-based platform designed for engineering teams to review 3D CAD models in real-time. It eliminates the need for heavy desktop software installations, allowing stakeholders to view, comment, and inspect designs instantly via a web link.

Whether you are performing a design review, client presentation, or internal audit, 3dverse keeps everyone on the same page—literally.

Key Features

⚡ Real-time Sync

See where other users are looking and pointing. Camera views and cursors are synchronized instantly.

🎙️ Voice Chat

Integrated voice communication. See who is speaking and mute/unmute directly in the workspace.

💬 3D Annotation

Pin comments directly onto the 3D model surface. Organize discussions by threads and resolve issues.

🤖 AI Assistant

Powered by Gemini. Summarize comments, critique views, and ask technical questions about the assembly.

🛠️ Inspection Tools

Explode views, measure distances, toggle wireframes, and cut sections to see internal details.

🌗 Dark Mode

A professional dark theme for low-light environments, automatically syncing the 3D viewport.

Getting Started

1. Creating a Project

To start, click the New Project button on the dashboard. You will be prompted to:

  • Enter a project name.
  • Upload a 3D file. Supported formats: .glb, .gltf, .obj, .fbx.

Note: Files are stored locally in your browser (IndexedDB) for security and speed in this version.

2. Sharing

Once inside a workspace, click the Share button in the top right. You can copy the link or invite users via email mock-up.

User Guide

Navigation

  • Rotate: Left-click and drag.
  • Pan: Right-click and drag.
  • Zoom: Scroll wheel.
  • Focus: Double-click on an object to center the camera.

Presentation Mode

Click the Play icon in the header to start presenting. All other users in the room will have their cameras locked to your view. Click Stop Presenting to release control.

Tools

The floating toolbar provides several inspection utilities:

  • Select (Cursor): Default mode for rotating and clicking objects to comment.
  • Explode (Layers): Separates the assembly components for better visibility.
  • Measure (Ruler): Click two points on the model to calculate the straight-line distance.
  • Section (Scissors): Enables a clipping plane. Use the slider to move the plane up and down.
  • Wireframe (Box): Toggles mesh wireframe rendering.

AI Design Assistant

The AI Assistant (Gemini Flash) can be accessed via the Sparkles icon in the toolbar. It is context-aware:

  • It knows the project status and parts list.
  • It can read the current comments.
  • It can "see" your current view (via screenshot capture) to analyze geometry.
Tip: Use the "Critique View" quick action to have the AI inspect the current camera angle for design flaws.

Settings & Theme

Access settings from the Dashboard sidebar. You can:

  • Update your profile (Name, Role, Bio).
  • Manage notification preferences.
  • Toggle between Light and Dark themes.

Theme changes persist across reloads and affect the UI as well as the 3D environment lighting.

Permissions

To use the voice chat features, the application requires access to your microphone.

When joining a workspace, your browser will prompt for permission: "3dverse wants to use your microphone." Please click Allow to enable voice features.

If you deny permission, you can still view and comment, but you will appear as muted to others.

Keyboard Shortcuts

Boost your productivity with these hotkeys inside the workspace:

Troubleshooting & FAQ

Why is my microphone not working?

Ensure you allowed browser permissions. Check the URL bar for a camera/mic icon to reset permissions. Also, verify your system's input device settings.

Supported File Sizes?

We recommend files under 50MB for optimal browser performance. Large textures or high-poly meshes (>500k polygons) may cause lag on lower-end devices.

Is my data secure?

In this version, files are stored locally in your browser's IndexedDB. Real-time events are simulated locally for demonstration. In a production deployment, data would be encrypted via WebSocket (WSS).
